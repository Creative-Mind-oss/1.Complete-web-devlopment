selectors in HTML

Element selector

The element selector in CSS is a selector that targets all instances of a specific HTML elements in a document.

Id selector

An ID selector in CSS targets a single, unique element on a webpage by using the element's id attribute. Since IDs are meant to be unique, the ID selector is useful for applying styles to a specific element without affecting others.

Class Slector

A class selector in CSS is used to style multiple elements that share the same class attribute. Its useful when you want to apply the same styling to a group of elements without affecting others, giving more flexibility than the id selector.
